Search for Twitter lists

The unofficial way, powered by Google search engine.

At the moment there's no way to search for a specific list in Twitter. Say I want to obtain a list of lists (ehm) about Linux: there's no API or public functionality to do that.

An easy workaround: search it with Google's advanced search as follows:

site:twitter.com inurl:lists <keyword>

The command will search within twitter.com site for URL paths containing lists the keyword keyword. For example:

site:twitter.com inurl:lists linux
